# Payout Management MVP — Frontend

Next.js frontend for the Payout Management system.

## Tech Stack
- Next.js (App Router)
- MUI (Material UI)
- Tailwind CSS
- Axios

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Create `.env.local` file:
   ```bash
   NEXT_PUBLIC_API_URL=http://localhost:5000
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

The app will start on `http://localhost:3000`.
